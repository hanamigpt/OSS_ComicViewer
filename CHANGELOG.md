# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-01

### Added

- PHP + SQLite based vertical-scroll comic CMS MVP.
- Single-user admin login.
- Series and episode management.
- Image, spacer, and text blocks.
- Public home, series page, and reader page.
- Basic upload validation and admin CSRF protection.
- README, SPEC, MIT License, and local setup instructions.
- Upload limit messaging for PHP `upload_max_filesize` and `post_max_size`.

### Notes

- This release is intended for local development and small self-hosted experiments.
- Public deployment requires additional security review.
